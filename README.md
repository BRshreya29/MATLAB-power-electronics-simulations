# MATLAB Power Electronics Simulations

This repository contains MATLAB/Simulink models and hardware-supported experiments focused on core power electronics concepts. The work includes circuit simulations, performance analysis, and validation through laboratory measurements.

## Contents

### **1. IGBT Characteristics**
Study of switching behaviour, rise/fall times, and gate-driver interfacing using TLP250.  
Includes:
- Simplified IGBT equivalent circuit  
- Pulse firing circuit  
- Hardware observation tables  

### **2. DC–DC Buck & Boost Converter**
MATLAB-only analysis of buck and boost converters.  
Includes:
- Duty cycle vs output voltage  
- Efficiency and transient response  
- Steady-state simulation waveforms

### **3. Lead-Acid Battery Charging (EV Application)**
Analysis of charging characteristics using an IGBT-based bidirectional DC–DC converter.  
Includes:
- Circuit diagram & block diagram  
- Converter efficiency evaluation  
- Battery charging characteristics  
- Final L, C, and η values for multiple duty cycles  

## Focus Area
All projects are centered on **Power Electronics**, combining:
- MATLAB/Simulink simulations  
- Hardware experiment support  
- Converter design and analysis
